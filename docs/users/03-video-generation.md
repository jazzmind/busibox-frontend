# Video Generation with Sora-2

## Access

Access video generation from the media app.

## Creating a Video

1. Enter a text prompt (max 500 characters)
2. Select duration: 4, 8, or 12 seconds
3. Choose resolution: HD, Full HD, vertical, or square

## Reference Media

Optionally upload reference media to guide the output:

- **Image**: JPG, PNG, or WebP. Max 10MB.
- **Video**: MP4 or MOV. Max 25MB.

Reference media provides visual context for the generation.

## Status Tracking

Generation progresses through states:

- **QUEUED**: Request accepted, waiting for processing
- **PENDING**: Processing slot allocated
- **PROCESSING**: Generation in progress (progress percentage shown)
- **COMPLETED**: Video ready for download

## Expiration

Videos expire 7 days after completion per OpenAI retention policy. Download before expiration.

## Video Library

Filter videos by:

- **My Videos**: Videos you created
- **Public**: Publicly shared videos
- **Shared**: Videos shared with you

## Sharing

Share videos with specific users. Set visibility to Private, Public, or Shared.

## Rate Limits

OpenAI rate limits apply. Typical limits are 5-20 requests per minute depending on account tier.
