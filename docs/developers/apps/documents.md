# Documents

**Package:** @busibox/documents  
**Port:** 3006  
**Base path:** /documents

## Purpose

Document libraries, upload, viewing, and chunk browsing. Manages document lifecycle from upload through processing and semantic search.

## Routes

Uses the same shell structure as portal, plus document-specific routes for libraries, uploads, and content viewing.

## Integration

- data-api: file upload and processing
- agent-api: search

## Features

- Library management (personal and shared)
- Document upload
- Status tracking
- Content viewing
- Chunk browsing
- Semantic tag groups

## Tech Stack

- Next.js 16
- Tailwind CSS 4
