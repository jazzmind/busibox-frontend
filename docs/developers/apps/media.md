# Media

**Package:** @busibox/media  
**Port:** 3005  
**Base path:** /media

## Purpose

Video generation with OpenAI Sora-2 and media management. Provides text-to-video generation and a video library.

## Routes

Uses the same shell structure as portal, plus media-specific routes for generation and library management.

## Features

- Text-to-video generation (OpenAI Sora-2)
- Reference media upload
- Status polling
- Video library (my videos, public, shared)
- Sharing
- Expiration management

## Tech Stack

- Next.js 16
- Tailwind CSS 4
