# Chat

**Package:** @busibox/chat  
**Port:** 3003  
**Base path:** /chat

## Purpose

Conversational AI interface using the busibox-app ChatPage component. Provides a focused chat experience with agent-api for LLM interactions.

## Routes

Uses the same shell structure as portal (`/(authenticated)/`, `/login`, etc.).

## Features

- Conversation management
- SSE streaming for responses
- Markdown, code, and math rendering
- Web search
- Document search
- Library selection

## Integration

Uses busibox-app ChatPage for conversations, message streaming, and insights panel. Integrates with agent-api for LLM interactions.

## Tech Stack

- Next.js 16
- Tailwind CSS 4
